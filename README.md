# JavaScript Foundation Projects

This repository contains 10 small JavaScript practice projects created to strengthen core JavaScript fundamentals.

The main focus is not building complex applications. The goal is to practice JavaScript step by step using simple, approachable projects.

## Projects

1. **Toggle Text Visibility**  
   Practice DOM selection, click events, show/hide logic, and button text updates.

2. **Like Counter**  
   Practice variables, number increment/decrement, click events, and UI updates.

3. **Add and Remove List Items**  
   Practice input values, arrays, creating elements, deleting items, and DOM updates.

4. **Form Validation**  
   Practice conditions, input validation, error messages, and form handling.

5. **Tip Calculator**  
   Practice input handling, basic calculations, functions, and displaying results.

6. **Character Counter**  
   Practice input events, string length, and real-time UI updates.

7. **Password Show / Hide**  
   Practice changing input types, event handling, and condition logic.

8. **Word Counter**  
   Practice strings, `trim()`, `split()`, word counting, and real-time updates.

9. **Text Case Converter**  
   Practice string methods, textarea input, click events, functions, and DOM updates.

10. **Search / Filter List**  
    Practice arrays, `filter()`, `includes()`, input events, and displaying filtered results.

## Learning Goals

- Build strong JavaScript fundamentals
- Practice DOM manipulation
- Understand events and user interactions
- Improve condition-based logic
- Work with strings, arrays, and functions
- Learn how to update the UI dynamically
- Build confidence for frontend interviews

## Tech Stack

- HTML
- CSS
- JavaScript

## Purpose

These are beginner-friendly JavaScript foundation projects. Each project is intentionally small and focused on one or two important JavaScript concepts.

The goal is to complete each project manually and understand how JavaScript works in real browser-based interactions.
